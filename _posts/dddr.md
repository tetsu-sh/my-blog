整理しつつ英語の勉強

- Entity
  属性ではなく ID で識別される
  ライフサイクルがあり、それを通して変化し得る
- value object
  属性とモデルの要素のロジックが重要。
  Immutable 　ミュータブルな状態に依存しない副作用のない操作
- ドメインイベント
  ドメインに関するイベントをドメインオブジェクトとする
- エンティティやバリューに責務を持たせられないプロセスはサービス、指きたすネームを与えろ
- モジュール。低結合にね。
- 凝集,集約.集約ルートを定め。ルートへの参照のみが外部からのアクセスを許す。
